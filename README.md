# Everquint Assignment

This project is a frontend application built as part of the Everquint technical assignment. It demonstrates modern web development practices, clean architecture, and efficient state and data handling.

---

## 🚀 Overview

The application focuses on delivering a responsive and user-friendly interface while maintaining scalable and maintainable code structure. It showcases key frontend concepts such as component-based architecture, API integration, and performance optimization.

---

## ✨ Features

- Clean and modular component structure
- Responsive UI design
- API integration and data handling
- Error handling and loading states
- Reusable components
- Optimized rendering and performance
- Scalable folder structure

---

## 🛠️ Tech Stack

- **Frontend:** React (with Hooks)
- **Language:** JavaScript / TypeScript (update if needed)
- **Styling:** CSS / Tailwind / Material UI (update accordingly)
- **State Management:** React State
- **DB** IndexedDB

---

## 📁 Project Structure

src/
│── components/ # Reusable UI components
│── pages/ # Page-level components
│── services/ # API calls and external services
│── hooks/ # Custom React hooks
│── utils/ # Helper functions
│── assets/ # Images, icons, etc.
│── App.js # Root component
│── index.js # Entry point

## ⚙️ Installation & Setup

1. Clone the repository:

git clone https://github.com/LNR14/EverquintAssignment.git

2. Navigate to the project directory:
   cd EverquintAssignment

3. Install dependencies:
   npm install

4. Start the development server:
   npm start
